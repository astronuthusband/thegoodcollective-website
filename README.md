# The Good Collective

🌐 https://thegoodcollective.my

## About the Project

**The Good Collective** is a modern digital marketing agency website designed and developed to represent a premium, conversion-focused online presence for a creative services business.

This project was built to showcase my skills in:
- Web design & UI structure
- Responsive front-end development
- Brand positioning for digital services
- Conversion-focused layout design
- Basic SEO structure and performance awareness

---

## 🎯 Purpose

The goal of this website is to:
- Present The Good Collective as a professional digital marketing agency
- Communicate services clearly to potential clients
- Build trust through modern design and structured content
- Convert visitors into leads through clean UX and call-to-actions

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive design principles
- Git & GitHub for version control
- Netlify for deployment

---

## 💡 Features

- Fully responsive layout (mobile, tablet, desktop)
- Clean and modern UI design
- Service-focused landing structure
- SEO-friendly HTML structure
- Fast-loading static site
- Contact/lead generation sections

## 🎨 Design Approach

The design follows a **minimal, premium agency style** inspired by modern digital studios.  
Focus was placed on:
- Clear typography hierarchy
- Strong spacing and layout balance
- Simple navigation flow
- High readability and conversion intent
